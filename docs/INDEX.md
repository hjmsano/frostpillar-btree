# Frostpillar BTree Docs

This repository provides a standalone, in-memory B+ tree component for JavaScript/TypeScript runtimes.

## Sections

- `README.md` / `README-JA.md`: user-facing usage and API guides (EN/JA)
- `docs/architecture/`: vision, architecture, testing principles
- `docs/specs/`: normative behavior specifications
- `docs/adr/`: architectural decision records
